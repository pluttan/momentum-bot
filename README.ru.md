# momentum-bot

**Спот-бот для торговли по моментуму на Binance с трейлинг-стопами и управлением через Telegram**

Спот-бот для торговли по моментуму на Binance. Каждый период удержания он ранжирует динамически обнаруженную вселенную ликвидных USDT-пар по смешанному моментум-скору (полная доходность + EWM-ускорение + всплеск объёма), покупает топ-N лучших и удерживает до следующей ребалансировки или срабатывания трейлинг-стопа. Включает React-дашборд с встроенным бэктестером, Telegram-бота для мониторинга и управления, а также оптимизацию параметров через Optuna. Поддерживает несколько параллельных торговых слотов, каждый со своим режимом, капиталом и переопределением параметров.

## ■ Возможности

- ❖ **Стратегия моментума** — окно 8 дней, период удержания 9 дней, топ-2 концентрированных ставки, смешанное ранжирование: полная доходность + EWM-ускорение + всплеск объёма
- ❖ **Трейлинг-стоп** — ~7% от максимума высокой отметки, поднимается вверх; ~-1% жёсткий пол на позицию
- ❖ **3 режима** — SCAN_ONLY (только ранжирование), PAPER (виртуальные сделки в SQLite), LIVE (реальные ордера на Binance)
- ❖ **Мульти-слот** — запуск нескольких PAPER/LIVE-экземпляров параллельно, каждый со своей БД, капиталом и переопределением параметров
- ❖ **Telegram-бот** — `/status`, `/positions`, `/pnl`, `/history`, `/params`, `/top`, `/pause`, `/resume`, `/stop`, `/reload`
- ❖ **Веб-дашборд** — React SPA (кривая эквити, аллокация, лог сделок) на FastAPI, со встроенным запуском бэктеста в браузере
- ❖ **Бэктестинг** — многолетняя историческая симуляция с оптимизацией параметров через Optuna (TPE)
- ❖ **Динамическая вселенная** — автообнаружение всех ликвидных USDT-спот-пар по объёму за 24ч, с опциональной фильтрацией по тегам Seed/Monitoring
- ❖ **Риск-ограничения** — MAX_DRAWDOWN_PCT (25% по умолчанию), дневной лимит потерь, без плеча, только спот
- ❖ **Скидка на комиссию BNB** — эффективная ставка комиссии 0.075%

## ■ Параметры стратегии

<div align="center">

| Параметр | Значение |
|-----------|-------|
| Вселенная | динамическая (все ликвидные USDT-спот, ≥$10M объём за 24ч) |
| Окно | 8 days |
| Период удержания | 9 days |
| Топ N | 2 |
| Стоп-лосс | ~-1% floor |
| Трейлинг-стоп | ~7% from HWM |
| Вес ускорения | 0.07 |
| Вес всплеска объёма | 0.30 |
| Комиссии | 0.075% (BNB) |

## ■ Запуск

```bash
# Установка (uv venv + editable deps)
make setup

# Ранжирование вселенной (без торговли)
make scan

# Бумажная торговля с уведомлениями в Telegram
make paper

# Веб-дашборд на :8787
make dashboard

# Запуск тестов
make test

# Бэктест
make backtest
```

Параметры стратегии находятся в `src/momentum/config.py` (дашборд может их перезаписывать).
Секреты передаются через переменные окружения: `BINANCE_API_KEY`, `BINANCE_API_SECRET`,
`TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`, `TELEGRAM_OWNER_ID`, `DASHBOARD_PASS`.
